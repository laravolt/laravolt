## 2024-05-11 - Optimized Permission Check
**Learning:** Laravolt's `HasRoleAndPermission` and `Role` models do dynamic query checks or `contains` lookups when `hasPermission` is called. Since permissions are eager-loaded, doing a `contains` operation manually can bypass `app(config(...))` overhead and Model instantiation.
**Action:** Overrode `_hasPermission` in models to utilize eager-loaded relations properly.

## 2024-06-19 - Bulk fetch relationship IDs before syncing
**Learning:** In Laravel, `firstOrCreate` inside a loop evaluating string names for many-to-many relationship syncing causes severe N+1 database queries.
**Action:** Always pre-fetch existing records in bulk by string name using `whereIn` and `keyBy` to resolve primary keys in-memory before looping through string lists.
